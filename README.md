# 🛒 Tienda Online en Python

Proyecto académico desarrollado en Python que simula una tienda online.

## 📌 Funcionalidades

- Registro de usuarios
- Inicio de sesión
- Visualización de productos
- Control de stock
- Carrito de compras
- Eliminar productos del carrito
- Vaciar carrito
- Finalizar compra
- Métodos de pago
- Generación de factura
- Historial de compras

## 🛠️ Tecnologías utilizadas

- Python 3
- Programación Orientada a Objetos
- Archivos de texto
